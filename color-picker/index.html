How to Use the Color Picker

Click the color input to open your browser's native color picker. Alternatively, type a HEX code (like #ff5733), RGB value (like rgb(255, 87, 51)), or HSL value (like hsl(11, 100%, 60%)) in the text field.

All formats update in real-time. Click any value (RGB, HSL, CMYK) to copy it to your clipboard, or use the copy icon next to the HEX field.
